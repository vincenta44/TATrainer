About the Web

This page is meant to reflect off of what I learned about the web and its basic languages (HTML, CSS, Javascript).
From the Marksheet articles linked in the portfolio description. I have a better understanding about what the interent is and how it works.
Here are the important details:

  • Client: A program that can take up many forms
  • Server: A dedicated computer always connected to the internet to deliver content
  • Protocol: A set of rules which computers use to communicate to eachother
  • Domain: A unique identifier for a computer connected to the internet
  • Web: Part of the internet that uses the HTTP protocol
  • Webpage: A document written in HTML
  • Website: A collection of webpages on the same domain

HTML

It is important to know the basic set up of a HTML document.
It is as follows:

            
<!DOCTYPE html>

    <html>

        <head>

            <meta charset="utf-8">

            <title>The title of the page</title>

        </head>

        <body>

            <h1>This is a header and they come in all sizes from 1-6</h1>

            <p>This is a paragraph</p>

        </body>

    </html>

Each tag has to have a beginning and end denoted by <p> which starts the tag and </p> which ends it.
Elements are defined by the starting tag and are components in an HTML document. Attributes are added in the opening tag of an element to change its behavior.
There are many elements and attributes to be used in the body of the webpage that can be used (not limited to):

Element Examples

  • h1: Element that creates a heading from the largest <h1> to the smallest <h6>

  • p: Element used to add a paragraph to add text

  • img: Element to display an image

  • ul: Element to create an unordered list and uses<li>for the list items.

Attribute Examples

  • href: Attribute used to define the target of the link

  • class: Attribute which specifies one or more class names for an element

  • id: Attribute that specifies a unique ID for an element

  • style: Attribute that specifies an inline CSS style for an element

There are also block elements such as <p> and inline elements such as <strong> that are used to distinguish text.

Lastly, I want to touch on <div> which is an element that divides a certain part of an HTML document and can be used to seperatley edit and style that section.

CSS

CSS stands for Cascading Style Sheets and is used to design and style an HTML file and can change colors of anything on the document, fonts of any text, and overall add style to a website.
CSS can be added within a websites HTML file or linked as a seperate style sheet. Linking a style sheet externally looks like this <link rel="stylesheet" href="styles.css">

CSS added to the HTML document can be seen as inline which is using the <style> element within the beginning tag or external which is using <style> in the head of the HTML document and referencing elements by name, class, or ID.
On this html page I use internal html in the head of the file. As an example, this is what it looks like: 

            
        .menu{
            padding:12px 14px;
        }
        .menu a{
            padding:12px 14px;
            color: black;
        }
        .menu a:hover{
            background-color: rgb(145, 143, 143);
            padding: 8px 10px;
            border-radius: 25px;
        }
        .top {
            text-align: center;
            text-decoration: underline;
        }
        h2{
            text-align: center;
            background-color: #eeeeee;
            width: 100%;
            border-radius: 10px;
        }
        pre {
            background: #eeeeee;
            border: 1px solid rgb(109, 109, 109);
            max-width: 45%;
            display: block;
        }
        code{
            font-size: 14px;
            border-color: #eeeeee;
            color: black;
            background-color: #eeeeee;
            padding: 4px 4px;
            border-radius: 10px;
        }
        .codeSnippet{
            border-color: white;
            color: black;
            background-color: transparent;
        }
        .res {
            background-color: rgba(255, 255, 255, 0.829);
            width: 40%;
            height: 100px;
            overflow: scroll;
        }

There are countless properties and attributes available with CSS that can help style any HTML file.

CSS is a three step process:

  • Selector: Defines who the target is
  • Property: Defines what charateristic to alter
  • Value: Defines how to alter the characteristic

Javascript

Javascript goes hand in hand with CSS and HTML and can generlaly be added internally to an HTML file by adding the <script> element.
Just like CSS, Javascript can be added internally or externally. In order to add a script file externally you must link it using the src attribute.
